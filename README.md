# Wtool
---------
### 我的工具集，有很多工具
---
##### 1. 它能做什么？
这是一个用Go语言实现的常用数据结构集合，为开发者提供高效易用的工具。

---
# 它有很多工具，例如：
## set（集合）, stack（栈）, queue（队列）, linkedList（链表）

### 可用的数据结构：

#### Stack（栈）
- `New`: 创建一个新的栈实例
- `Push`: 将元素添加到栈顶
- `Pop`: 弹出并返回栈顶元素
- `Peek`: 返回栈顶元素但不移除
- `Len`: 获取栈中元素的数量
- `Empty`: 检查栈是否为空
- `Clear`: 清空栈中的所有元素

#### Queue（队列）
- `New`: 创建一个新的队列实例
- `Enqueue`: 将元素添加到队列尾部
- `Dequeue`: 弹出并返回队列头部元素
- `Front`: 返回队列头部元素但不移除
- `Back`: 返回队列尾部元素但不移除
- `Len`: 获取队列中元素的数量
- `Empty`: 检查队列是否为空
- `Clear`: 清空队列中的所有元素

#### Set（集合）
- `New`: 创建一个新的集合实例
- `Add`: 向集合中添加元素
- `Remove`: 从集合中移除元素
- `Has`: 检查元素是否存在于集合中
- `Len`: 获取集合中元素的数量
- `Clear`: 清空集合中的所有元素
- `Items`: 以切片形式获取所有元素

#### LinkedList（双链表）
- `New`: 创建一个新的链表实例
- `PushFront`: 将元素添加到链表前端
- `PushBack`: 将元素添加到链表后端
- `PopFront`: 弹出并返回前端元素
- `PopBack`: 弹出并返回后端元素
- `GetLast`: 获取最后一个元素但不移除

### 命名约定：
该项目在所有数据结构中遵循一致的命名约定：
- 构造函数统一命名为 `New`，使用可变参数进行初始化
- 大小相关的方法统一命名为 `Len`
- 空值检查方法统一命名为 `Empty`
- 添加/删除操作使用描述性名称，如 `Push`、`Pop`、`Enqueue`、`Dequeue`、`Add`、`Remove`
- 访问器方法使用简短清晰的名称，如 `Front`、`Back`、`Peek`、`Has`
